# Presentation: TDD with buster.js

Slides/code for JavaScript Latvia meetup on 2014-01-14

## View
* `git clone` this
* `npm install`
* `npm start`
* Open http://localhost:9001/
* In the editors on the slides:
	* Shift+Click - load "starter" file (fill in the blanks...)
	* Cmd+S/Ctrl+S - send current content to the server
	* Ctrl+L - load final file
	* Shift+Tab - switch between code/tests
	* Ctrl+[ / Ctrl+] - navigate slides

## Develop
* `grunt watch` and/or `npm test`
